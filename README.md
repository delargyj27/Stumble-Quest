# Stumble-Quest

## Development


### Prerequisites
- Disable `display_errors` in PHP
- Enable `log_errors` in PHP

### Run the app

1. Run AMPPS
2. Open http://localhost/Stumble-Quest/index.php

### View PHP Error Logs

1. Open AMPPS log file

        C:\Ampps\apache\logs\error.log

### Compile Tailwind CSS

```
npm run dev
```
